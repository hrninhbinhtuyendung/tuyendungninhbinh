# Tuyen dung Ninh Binh - React + Vite + Supabase

## Chay du an

```bash
npm install
npm run dev
```

## Cau hinh Supabase

1. Copy file `.env.example` thanh `.env`.
2. Dien 2 bien moi truong:

```env
VITE_SUPABASE_URL=https://your-project-ref.supabase.co
VITE_SUPABASE_ANON_KEY=your-anon-key
```

3. Trong Supabase SQL Editor, chay script:

```sql
create table if not exists public.jobs (
  id bigint generated by default as identity primary key,
  title text not null,
  company text not null,
  salary text not null,
  location text not null,
  tags text[] default '{}',
  description text,
  created_at timestamptz not null default now()
);
```

## Da tich hop

- Trang chu (`/`) doc danh sach viec lam tu bang `jobs`
- Trang dang tuyen (`/post`) tao tin tuyen dung moi len Supabase
- Trang chi tiet (`/job/:id`) lay chi tiet 1 cong viec tu Supabase
